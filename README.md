Aplicação voltada para Fábrica de Ovos.
Esta aplicação possui as seguintes funcionalidades:

## Lote de Produção
## Fábrica de Ração
## Controle de Produção
## Controle Financeiro (Contas a Pagar e Receber)
## Consumo de Ração
## Ovos Produzidos
## Controle de Mortalidade
